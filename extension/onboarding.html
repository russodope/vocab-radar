📖
浏览英文网页时划词查询，DeepSeek 给上下文感知翻译

不认识的词自动高亮，下次读到同一篇就能"复习到"
🎯
查询 3 次以上的"卡词"用更深的橙色，提醒重点关注
🔒
所有数据存在你浏览器本地（IndexedDB），不上任何服务器

1. 准备 DeepSeek API Key

  1. 访问 platform.deepseek.com/api_keys 登录
  2. "Create new API key" 创建一个新 key（命名为 VocabRadar 方便管理）
  3. 复制 key（形如 sk-xxxxxxxx）
  4. 建议：在 额度管理 给账户设个月度限额，比如 ¥10/月（重度使用一个月也就 ¥1-3）

2. 粘贴到下面，点测试
